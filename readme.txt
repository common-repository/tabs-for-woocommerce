Create product tabs and add product tabs in woocommerce product pages  

== Description ==
This plugin will help you create / edit / delete product tabs in product tbas WYSWYG editor.

When you edit product, please click "product tabs" link in product metabox, then select product tabs from tab list
 
In front end, selected tabs will show in woocommerce product page

== Installation ==

1:Upload the Tabs for WooCommerce to your blog
2:Activate it 
3:In wordpress admin area, you will find "Product Tabs" menu item
4:Click "Product Tabs" menu item, you can add / edit / delete product tabs
5:When edit woocommerce product, you will find  "product tabs" link in product metabox, please select product tabs from tab list
6:In front end, selected tabs will show in woocommerce product page
